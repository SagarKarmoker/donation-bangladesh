What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document as a tree of nodes. Each node represents part of the document, such as an element, attribute, or text node.

How do you select an element from the DOM?

There are several ways to select an element from the DOM. You can use methods like getElementById, getElementsByClassName, getElementsByTagName, querySelector, and querySelectorAll to select elements by their id, class, tag name, or CSS selector.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique for handling events in the DOM by attaching a single event listener to a parent element, rather than attaching multiple event listeners to individual child elements. This is useful because it reduces the number of event listeners in the document, which can improve performance and reduce memory usage.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes and styles using the DOM by accessing the element's properties and methods. For example, you can set an element's attribute using the setAttribute method, and you can set an element's style using the style property.
